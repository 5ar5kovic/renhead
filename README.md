# Renhead
## installation Instructions
1. Clone the repository;
2. Navicate to the directory and run:
~~~
composer install
~~~
3. Create the database, set up the .env file and run:
~~~
php artisan migrate --seed
~~~
4. Set up a virtual host for the application;
4. Let's Rock!
